# Archive Notice

This repository has been archived and is no longer actively maintained.

## New Location

The Forthic project has moved to the Forthix organization with individual repositories for each runtime:

**Index/Overview:** https://github.com/forthix/forthic

**Individual Runtimes:**
- Python: https://github.com/forthix/forthic-py
- TypeScript: https://github.com/forthix/forthic-ts
- Ruby: https://github.com/forthix/forthic-rb

## What This Means

- This repository is read-only
- No new issues or pull requests will be accepted
- No new releases will be published from this repository
- All published packages will continue to work

## Migration Guide

### For Users

If you're using published packages:

- **NPM (@forthic/interp):** Packages will continue to work. Watch https://github.com/forthix/forthic-ts for updates.
- **PyPI (forthic):** Packages will continue to work. Watch https://github.com/forthix/forthic-py for updates.
- **RubyGems (forthic):** Packages will continue to work. Watch https://github.com/forthix/forthic-rb for updates.

### For Contributors

Please direct your contributions to the appropriate repository:

- **Python runtime issues/PRs** → https://github.com/forthix/forthic-py
- **TypeScript runtime issues/PRs** → https://github.com/forthix/forthic-ts
- **Ruby runtime issues/PRs** → https://github.com/forthix/forthic-rb
- **General questions/discussions** → https://github.com/forthix/forthic

## Contact

For questions about this transition, please open an issue in the relevant repository:
- General: https://github.com/forthix/forthic/issues
- Python-specific: https://github.com/forthix/forthic-py/issues
- TypeScript-specific: https://github.com/forthix/forthic-ts/issues
- Ruby-specific: https://github.com/forthix/forthic-rb/issues
